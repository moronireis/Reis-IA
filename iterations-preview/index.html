Home MVP / Iteration 01 / 2026-04-15

Três direções de arte para a home. Uma será produção.

Cada direção é uma tese criativa completa, não uma variação. O objetivo desta revisão é uma decisão estética real — qual linguagem visual carrega a autoridade da REIS [IA] com mais nitidez na primeira frame.

Dir 01
Mercury · Pentagram · Stripe

Editorial Gravity

A home composta como um ensaio financeiro. Tipografia em escala de revista, grid assimétrico, disciplina suíça. A ausência de movimento é o movimento.

"Reads like an FT editorial printed on obsidian."

Dir 02
Linear · Raycast · Cursor

Architectural Precision

A home como uma ferramenta profissional que por acaso é página de marketing. Densidade calibrada, monoespaço como co-protagonista, spotlight radial seguindo o cursor, state-machine demo do método.

"Feels like opening a Raycast prompt in a room that was waiting for you."

Dir 03
Arc · Apple · Bloomberg Graphics

Cinematic Gravity

A home como um filme. Gradient mesh WebGL na abertura, cortes cinemáticos entre capítulos, capítulo de método como reveal tipográfico pinado, proof em estilo Bloomberg feature.

"Scrolls like a Koyaanisqatsi cut. Holds like a Bloomberg feature."
