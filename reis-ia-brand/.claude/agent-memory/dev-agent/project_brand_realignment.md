"Time Builders" has been renamed to "Builders" across all pages. Z7 (symbol, Z7 Days product, Z7 Hours, Z7 Months, Z7 philosophy) is permanently removed from the brand.

**Why:** Brand simplification. The page concept shifted from "Time Builders Movement + Z7 Days program" to "Builders — community for learning to build and sell AI agent projects."

**How to apply:**
- Never use "Time Builders" in any copy, heading, CTA, or component — always "Builders"
- Never import or use Z7Icon component — it is deleted
- Never import or use MockupZ7Timeline component — it is deleted
- Never reference "Z7 Days", "Z7 Hours", "Z7 Months", or "Z7 philosophy"
- The "7 dias = 7 meses" compression framing is removed (was tied to Z7)
- CTAs read "Aplicar para Builders" (not "Time Builders")
- Founder section: "O Primeiro Builder" (not "Time Builder")
- Program references: "programa Builders" or "mentoria Builders"
- CSS class `.builder-z7-meaning` is removed — do not recreate
