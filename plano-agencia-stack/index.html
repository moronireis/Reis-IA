05Sistema multi-cliente

Single source of truth: ACTIVE_CLIENT.md na raiz. Arquivo markdown simples, legível por qualquer agente sem tooling especial.

Preâmbulo obrigatório (todo agente, toda task)

1. Read /ACTIVE_CLIENT.md
2. Extract client_slug
3. If <none> or missing → HALT, pedir ativação
4. Load brain/clients/{slug}/CLIENT.md (business context)
5. Load voice profiles em brain/clients/{slug}/voice-profiles/
6. Load design tokens em brain/clients/{slug}/design-system/
7. Escopar writes para brain/clients/{slug}/assets/

Estrutura CLIENT.md

# Client: {{CLIENT_NAME}}

slug: acme-corp
industry: SaaS B2B
stage: seed / growth / enterprise
language: pt-BR
primary_url: https://acme.com
founder: Jane Doe
ICP: [descrição]
offer_tier: high-ticket / low-ticket / mixed
active_pillars: [copy, social, design, video, ads]

Onboarding de novo cliente

  1. User: "Onboard client: acme-corp"
  2. Orchestrator roda bash .claude/templates/new-client-scaffold.sh acme-corp
  3. Script copia brain/clients/_TEMPLATE/ → brain/clients/acme-corp/
  4. Orchestrator pergunta: nome, indústria, stage, idioma, ICP, URLs
  5. Preenche CLIENT.md
  6. Delega a market-research-analyst (pesquisa) + brand strategists (voice profiles)
  7. Atualiza ACTIVE_CLIENT.md
  8. Cliente pronto para receber tasks

Trocar cliente mid-session

"Ativar cliente acme-corp" → orchestrator atualiza ACTIVE_CLIENT.md. Próxima task já escopa pro novo folder. Memórias dos agentes ficam transversais (aprendizados genéricos); assets são sempre por cliente.

Design library compartilhada

brain/shared/design-library/ é compartilhada entre todos os clientes. Harvests de Stripe, Linear, Apple valem para qualquer projeto. Mood reports específicos de cliente ficam em brain/clients/{slug}/design-system/mood-reports/.

06Script bash de cópia

Salvar como /Users/moronireis/create-agency-stack.sh. Não executar ainda.

#!/usr/bin/env bash
set -euo pipefail

SRC="/Users/moronireis/Projetos vscode"
DST="/Users/moronireis/agencia-ia-stack"

if [[ -d "$DST" ]]; then
  echo "ERROR: $DST already exists. Aborting."
  exit 1
fi

echo "Creating stack skeleton at $DST ..."
mkdir -p "$DST"/{.claude/{agents,rules,voice-profiles,templates,agent-memory,scripts},brain/{shared/{frameworks,design-library,context,research},clients/_TEMPLATE/{voice-profiles,research/{market,competitors,audience},strategy,messaging,branding,assets/{copy,campaigns,content,design,video},design-system,backups}},scripts/video-pipeline}

# Copy agents
rsync -av --exclude='.DS_Store' \
  "$SRC/.claude/agents/" "$DST/.claude/agents/"

# Copy global rules
cp "$SRC/.claude/rules/hormozi-framework.md"      "$DST/.claude/rules/"
cp "$SRC/.claude/rules/humanization-rules.md"     "$DST/.claude/rules/"
cp "$SRC/.claude/rules/brand-voice.md"            "$DST/.claude/rules/brand-voice-template.md"
cp "$SRC/.claude/rules/brand-audit-checklist.md"  "$DST/.claude/rules/"

# Copy ollama script
cp "$SRC/.claude/scripts/ollama-query.sh" "$DST/.claude/scripts/"
chmod +x "$DST/.claude/scripts/ollama-query.sh"

# Seed voice-profile template
cp "$SRC/.claude/voice-profiles/moroni-personal.md" \
   "$DST/.claude/voice-profiles/_TEMPLATE.md"

# Design library (shared asset)
rsync -av --exclude='.DS_Store' \
  "$SRC/brain/design-library/" "$DST/brain/shared/design-library/"

# Video pipeline
cp "$SRC/brain/context/video-pipeline-oss-recipe.md" \
   "$DST/brain/shared/context/" 2>/dev/null || true

# Empty agent-memory stubs (one per agent)
for agent_file in "$DST"/.claude/agents/*.md; do
  agent_name=$(basename "$agent_file" .md)
  mkdir -p "$DST/.claude/agent-memory/$agent_name"
  cat > "$DST/.claude/agent-memory/$agent_name/MEMORY.md" <<EOF
# $agent_name Memory

Last updated: (empty — fresh stack)

## Patterns
(none yet)

## Feedback
(none yet)
EOF
done

# ACTIVE_CLIENT.md
cat > "$DST/ACTIVE_CLIENT.md" <<'EOF'
# Active Client

client_slug: <none>
activated_at: (none)
activated_by: (none)
EOF

# .gitignore
cat > "$DST/.gitignore" <<'EOF'
.DS_Store
node_modules/
dist/
.vercel/
.next/
.astro/
__pycache__/
*.pyc
venv/
.venv/
.env
.env.local
*.log
EOF

# Purge leftover .DS_Store
find "$DST" -name '.DS_Store' -delete 2>/dev/null || true

echo "DONE. Stack skeleton created at: $DST"
