Brand mark refinement Step A completed 2026-03-17, producing 5 subtle variations of the H1-B "Open Intersection" Hourglass plus 6 size-optimized variants.

**Variation A (Recommended)**: Recentered intersection at y=32, wider gap (8 units vs original ~5.8), bars at y=8/56 for maximum canvas use.

**Size optimization system**: Stroke weight scales inversely with target size — 3px at 16px, 2.25px at 24px, 1.75px at 32px, 1.5px at 48-64px, 1.25px at 128px+. Gap width also adjusts wider at small sizes.

**Why:** The original mark's intersection gap was too narrow to read clearly at 16px favicon size. Reference analysis showed Apple uses optical size variants, Porsche uses precise centering, and all premium brands maintain mark clarity across all sizes.

**How to apply:** When implementing the hourglass in the website, use the size-optimized SVG variant that matches the rendering context. The favicon should use opt-16, nav should use opt-24, etc.
