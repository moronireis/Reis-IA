Per tempo-e-rei-brand-philosophy.md v3.0, these are PERMANENTLY REMOVED:
- Z7 symbol, code, products (Z7 Hours, Z7 Days, Z7 Months)
- 7 zones of transformation framework
- Super Powers of Time Builders narrative
- "Time Builders" name (replaced by "Builders")
- Compression 7:7 metric
- Hourglass as mandatory symbol (now decorative optional only)
- Prototype Graveyard as brand concept (can appear in copy if natural)

Also prohibited per CLAUDE.md: Gold, amber, terracotta, chess pieces, crowns, gradient text, azure whisper effects, emojis in UI, SaaS pricing patterns.

**Why:** Brand went through major simplification. Old elements created confusion.
**How to apply:** Never reference Z7, Time Builders, chess, gold in any creative brief. Hourglass = optional watermark at best.
