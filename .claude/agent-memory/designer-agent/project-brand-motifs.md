## Core Rule
Both motifs must appear on every page at least once (per CLAUDE.md).

## Motif Narrative Territories
- **Chess (Strategy)**: Builder, Partners, methodology/framework sections, competitive positioning
- **Hourglass (Time)**: Systems, urgency/footer CTAs, agitation sections, efficiency/metrics

## Established Sizes and Contexts
| Size | Usage | Opacity |
|------|-------|---------|
| 20-24px inline | Beside text, inline with copy | 100% accent or 40% white |
| 32-40px section icon | Card icons, section headers | 100% accent gold |
| 80px featured | Hero decorative beside headline | 100% accent gold |
| 200-300px watermark | Background behind CTA sections | 3-5% opacity |
| 200-250px hero bg | Large silhouette in hero | 5-8% opacity |

## Rule: Never in Same Visual Cluster
Chess and hourglass should not appear as visual icons within the same card or visual group. They serve different narratives. The Final CTA sections combine both in copy but only one appears as a visual element (typically hourglass as watermark).

**Why:** Visual co-location weakens each motif's individual narrative. Each should anchor its own section.
**How to apply:** When a section references both themes in copy, choose the dominant visual motif based on the section's primary narrative purpose.
