## ZNITH Client Brand — Key Decisions

**Client:** ZNITH, founded by Leilaine Campioto Messias
**Deliverable:** `/Users/moronireis/Projetos vscode/znith-brand/branding/company-znith-concept.md`
**Version:** v1.0 created 2026-04-08, all 17 elements complete

### Architecture Model
- Endorsed sub-brands (like Marriott/Virgin)
- Master: ZNITH → ZNITH.AI, ZNITH Consulting, ZNITH.AI OS (flagship), Lideres Leoes (movement), Z-PAY (future)
- Leilaine Campioto as connective tissue personal brand (not subordinate to corporate)
- Sub-brands do NOT endorse each other — cross-refs go through master brand

### Positioning
- Category: "Commercial Growth Architecture with AI"
- Word ZNITH owns: "architecture"
- Competitive map axes: Traditional Consultancy ↔ AI-Integrated Operations (X) × Tactical/Tool-Level ↔ Architectural/System-Level (Y)
- ZNITH occupies top-right (architectural + AI-integrated)
- Tagline: "Direção. Consciência. Legado."

### Color Architecture
- Navy #091022 governs (background foundation)
- Gold #DF9F3E signs (max 15% surface, never background fill)
- ZNITH.AI adds Tech Blue #2A5090
- ZNITH Consulting uses navy/graphite only (restraint)
- Lideres Leoes is ONLY context where gold is active accent (#FFD161)
- Leilaine personal: warm gold #E8A847

### Typography
- Cinzel (serif) for headings, authority, brand name
- Montserrat (sans) for body, UI, navigation
- NOT Inter (that's Reis IA's font)

### Key Differentiator
- ZNITH combines: strategic diagnosis + commercial architecture + AI implementation + ongoing governance
- Nobody else combines all four layers in Brazilian market

**Why:** This is a premium Reis IA client project. Existing brandbook and design system were already partially built — this concept document consolidates the strategic framework.

**How to apply:** Reference this when doing any future work on ZNITH brand materials, ensuring consistency with the endorsed architecture model and the navy+gold palette system.
