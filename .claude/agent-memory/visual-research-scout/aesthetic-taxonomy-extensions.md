## Tags validated in casamento-moroni-daphine mood report

All core tags from the agent definition were sufficient for this project.
No new tags needed to be coined — the existing taxonomy covered all four directions.

Notable pairing observations:

- `editorial-fine-art` + `romantic-film` = the most productive tension for personal/event
  sites. Structure from fine-art, emotional warmth from film. Neither alone is complete
  for a human event.

- `typography-as-hero` + `image-as-monolith` = almost always appear together in premium
  sites. They are complementary, not competing — the image holds the viewport,
  the type holds the emotion.

- `luxury-whisper` is almost always implemented through subtraction, not addition.
  The "whisper" is always something removed — a gradient, a shadow, a color, a CTA button.
  When curating, look for what is NOT there, not just what is.

- `motion-restraint` in wedding context means: transitions above 800ms feel editorial,
  below 300ms feel like SaaS. The transition duration itself carries aesthetic meaning.

## Tags that almost earned a place but didn't (watch for recurrence)

- `ceremonial-pacing` — the idea that scroll speed itself can be controlled to create
  ritual/ceremony. Seen in Lara Hotz (1200ms entry) and Studio Freight (hold-and-release).
  Not coined yet — needs to appear in 2+ projects before earning a slot in the taxonomy.

- `substrate-warmth` — the technique of using off-white/cream backgrounds to warm all
  photography without filters. Seen in Kinfolk (#F4EFE6), Anthropologie (#F9F6F3),
  Cereal. The "substrate" (background color) does photographic work. Strong candidate
  for a tag if it recurs in non-wedding editorial projects.
