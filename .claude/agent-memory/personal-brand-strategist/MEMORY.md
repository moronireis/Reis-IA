# Personal Brand Strategist — Memory

## Key Decisions (v1.0 — 2026-03-18)

### Archetypes
- **Primary**: Strategist (Sage + Ruler hybrid) — sees the board, makes calculated moves
- **Secondary**: Builder (Creator) — ships, constructs, shows work in progress
- **Shadow**: Impatient Judge — frustration with hype, shallow claims, wasted potential

### Core Identity Elements
- **Tagline**: "Start with the money." (personal) / "Strategy before technology." (institutional)
- **Character tension**: Generosity (open frameworks) vs. Selectivity (limited direct access)
- **Signature color**: Terracotta #C47A5A — uniquely Moroni's, no other brand layer uses it
- **Personal palette**: Warm White #F5F0E8 / Charcoal #1A1A1A / Soft Blue #6AADFF / Terracotta #C47A5A

### Content Pillars (4)
1. Revenue-First AI Implementation (methodology in action)
2. AI Business Landscape (market intelligence, data-driven analysis)
3. Agency Transformation (selling/delivering AI — Builder pipeline)
4. The Builder's Journey (personal narrative, build in public)

### Tribe Levels
- Observadores → Jogadores → Construtores → Parceiros → Estrategistas
- Advancement through action, not purchase

### Key Mantras
1. "AI is not a technology project. It is a revenue strategy."
2. "Start with the revenue, work backward to the tech."
3. "Stop prototyping. Start profiting."
4. "The window is open. It won't stay open."
5. "The board rewards the player who moves first — and moves right."

### Anti-Moroni Definition
- The AI Guru: rebranded from crypto, sells excitement, hoards knowledge, promises ease, values reach over respect, culturally rootless
