Noiva S.A. is a Reis IA client project. Wedding planning agency owned by Giovanna Paola (8 years, 3-person team, under R$500K revenue).

**Why:** Current sales process is broken — PDF with prices sent before value is established, Giovanna personally closes all sales (1.5h meetings), no content strategy warming leads, expectation mismatch when different assessor delivers.

**Key strategic decisions (April 2026):**
- Transition Giovanna from sales closer to content creator (highest leverage)
- Hire/train dedicated saleswoman to run 45-min structured meetings
- Eliminate PDF-first pricing — replace with quiz + nurture + meeting sequence
- 4 content pillars: Metodo (30%), Bastidor (25%), Educacao (25%), Prova Social (20%)
- 3 lead magnets: Quiz "Qual Assessoria", Checklist "50 Coisas", Mini-curso "Casamento sem Drama"
- Video specifically for grooms (2 min, practical tone, money-focused)
- Follow-up cadence: D+1, D+3, D+7, D+14, D+30
- 90-day implementation plan in 3 phases (quick wins → infrastructure → scale)

**How to apply:** Reference `brain/assets/campaigns/noiva-sa-funnel-strategy.md` for full strategy. Brand context in `noiva-sa-platform/brandbook/`. Products: Assessoria Inicio (R$3.1K), Completa (R$4.5K, carro-chefe), Exclusiva (R$7.5K).
