## Patterns
- Builder products use metallurgical metaphors (forge, tempering, smelting)
- Systems products use three suffixes: Engine (multi-component), Agent (single-function), Protocol (diagnostic/strategic)
- All product names stay in English even in PT-BR marketing contexts
- Support tiers use Portuguese names: Vigilia Estendida, Otimizacao Ativa, Comando de Operacoes
- For high-ticket B2B consulting: "OS" framing > "consultoria" framing (justifies premium pricing)
- Phase names as single PT-BR words work well for C-level (LEITURA, PROJETO, OBRA, AJUSTE, GOVERNO)
- Frequency-based support naming (Mensal/Semanal/Integrado) avoids quality-tier hierarchy
