Brand v3.0 was integrated into index, strategy, and guidelines pages on 2026-03-20.

**Key concepts:**
- "O Tempo e Rei" = central philosophy with dual meaning (Tempo e Rei / Reis surname)
- Z7 = Z (last letter, end) + 7 (perfection, completude) = hourglass silhouette
- Hierarchy: Tempo -> Eficiencia -> Receita (non-negotiable order)
- 4-layer ecosystem: Reis IA (#4A90FF), Moroni Reis (#6AADFF), Time Builders (#2D7AFF), Systems (#00B4FF)
- Archetypes: Governante/Heroi/Rebelde for Moroni
- Villains: Falsos Profetas, Hype, Tech-First, "Espera Pra Ver"

**Why:** These concepts replaced the old v1.0 positioning (generic AI consultancy). The audit at `brain/assets/audit-pre-phase6.md` identified index, strategy, guidelines as needing v3.0 updates.

**How to apply:** Any new brand-related page should reference these concepts. Strategy/guidelines are now PT-BR with v3.0 content. Check the audit for remaining items (7 agents still need updating, amber references in branding docs).
