Brief C — Reis IA Design System

Ampulheta H1-B
Interações VFX

8 efeitos visuais para a Variation A da ampulheta H1-B. Cada efeito é independente, auto-contido e pronto para integração no site. Paleta exclusiva #4A90FF / #2D7AFF / #00B4FF.

Efeito 01 / 08

Hover Glow Pulsante

Padrão heartbeat: 2 pulsos rápidos (0.15s cada) seguidos de pausa (0.4s), em loop. O box-shadow azul cresce e diminui em sincronia com o ritmo cardíaco. Ativa apenas no hover — sem animação quando o cursor não está sobre o elemento.

Trigger: hover Duração: 0.7s/ciclo Loop: infinito Método: CSS keyframes
Uso: ícone de nav, hero badge, CTA icon

Passe o cursor sobre a ampulheta

Efeito 02 / 08

Watermark Breathing

Respiração sutil de 8 segundos: scale 1.0 → 1.02 → 1.0 com ease-in-out. Ampulheta posicionada como marca d'água em 3–4% de opacidade no background de seções ou cards. O movimento é quase imperceptível — cria vida sem distrair o conteúdo principal.

Trigger: auto (loop) Duração: 8s Escala: 1.0 → 1.02 Opacidade: 4%
Uso: background de seções, hero watermark

Efeito 03 / 08

Hero Particle Formation

Canvas 2D com 60 partículas azuis que partem de posições aleatórias e convergem para desenhar as 5 linhas da ampulheta H1-B em ~2 segundos. Cada partícula viaja com easing suave até sua posição final. Partículas iniciam com variação de velocidade para criar fluidez orgânica.

Trigger: auto + click-to-replay Duração: ~2s formação Partículas: 60 Método: Canvas 2D + rAF
Uso: hero section, loading state, intro animado
Efeito 04 / 08

Line-by-Line Construction

Cada uma das 5 linhas da ampulheta aparece sequencialmente usando SVG stroke-dashoffset. Delay de 200ms entre linhas. Total de ~1.5s para construir o ícone completo. Ordem: barra top → diagonal esq → diagonal dir sup → diagonal dir inf → barra bottom.

Trigger: click replay Duração total: ~1.5s Por linha: 350ms Método: stroke-dashoffset CSS
Uso: hero entrance, page load, reveal on scroll
Efeito 05 / 08

Scroll Fill Progression

A opacidade da ampulheta é mapeada ao progresso de scroll do container. No topo: 15% de opacidade (quase invisível). No final: 100% (totalmente visível). O ícone "emerge" conforme o usuário rola a página — reforça a metáfora do tempo que passa.

Trigger: scroll event Opacidade: 15% → 100% Método: JS scroll listener Performance: throttled
Uso: hero fixo, seção de metodologia, parallax

Efeito 06 / 08

Easter Egg — 7 Cliques

No sétimo clique: a ampulheta gira 360° (1s), 12 partículas azuis explodem radialmente e desaparecem em 2s, e o toast "O Tempo é Rei" aparece por 2 segundos. Contador exibido abaixo do ícone. Após o evento, o contador reseta.

Trigger: 7 cliques Spin: 360° em 1s Partículas: 12 Toast: 2s
Uso: hidden delight, brand storytelling, retenção
Cliques: 0 / 7
Efeito 07 / 08

Aurora Temporal

Três orbs de radial-gradient (#4A90FF, #2D7AFF, #00B4FF em 6–10% de opacidade) orbitam lentamente ao redor da ampulheta em ciclos de 18s, 20s e 25s com filter blur. Cada orb tem uma trajetória elíptica independente. O efeito cria uma aura atmosférica premium ao redor do ícone.

Trigger: auto (loop) Ciclos: 18s / 20s / 25s Opacidade: 6–10% Método: CSS transforms + filter blur
Uso: hero principal, seção de destaque, about section

Animação automática — 3 orbs em órbita

Efeito 08 / 08

Scanner Line

Linha horizontal fina (#4A90FF a 30% de opacidade) percorre verticalmente de cima para baixo em um loop de 6 segundos. Gradiente linear de transparência nas extremidades para entrada/saída suave. Efeito de scan tecnológico que reforça a identidade de precisão e análise da Reis IA.

Trigger: auto (loop) Duração: 6s/ciclo Opacidade: 30% Método: CSS pseudo-element + keyframes
Uso: ícone de loading, card hover state, data visualization

Animação automática — 6s loop
