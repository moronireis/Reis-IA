Performance Patterns

GPU acceleration, paint containment, passive scroll listeners, and reduced motion support.

GPU Acceleration

will-change + translateZ(0)

Force GPU layer creation for animated elements.

.animated-element { will-change: transform, opacity; transform: translateZ(0); backface-visibility: hidden; }

Paint Containment

contain: paint

Isolate complex sections to prevent sibling repaints.

.paint-contained { contain: paint; }

Reduced Motion

prefers-reduced-motion

Respect user accessibility preferences.

@media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; scroll-behavior: auto !important; } }

Passive Scroll + rAF

{ passive: true } + requestAnimationFrame

Apple always uses passive listeners and rAF throttling.

let ticking = false; window.addEventListener('scroll', () => { if (!ticking) { requestAnimationFrame(() => { // update ticking = false; }); ticking = true; } }, { passive: true });
