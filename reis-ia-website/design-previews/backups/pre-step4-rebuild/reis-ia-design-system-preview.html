Reduced Motion
prefers-reduced-motion: reduce
When enabled, all animations are reduced to 0.01ms duration and scroll-behavior is set to auto. This is applied globally via CSS media query.
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
