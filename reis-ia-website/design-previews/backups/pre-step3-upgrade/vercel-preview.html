Signature Elements

1. The Black Canvas
Vercel's defining characteristic: pure black (#000000) backgrounds with depth created through subtle value shifts rather than shadows. Only 3-4 shades of near-black create the entire elevation hierarchy.
#000
#0A0A0A
#111
#1A1A1A
Reis IA Adaptation
Already aligned. The key learning is restraint -- use only 3-4 background values (#000, #0A0A0A, #111, #1A1A1A). More values dilute the premium feel. Add warmth through gold accent borders and hover states rather than warm-tinted backgrounds.
2. Minimal Elevation (Border + Background, Not Shadows)
In dark mode, depth is communicated through background value shifts and border visibility rather than traditional box-shadows. Shadows are reserved only for floating elements.
Flat
Raised
Float
Modal
Reis IA Adaptation
Use background-value shifts for card elevation. Add a subtle gold (#C9A84C at 20% opacity) border on premium card variants for brand differentiation. Reserve box-shadow for modals and dropdowns only.
3. Saturated Backdrop Blur
The navigation combines saturate(180%) with blur(20px). The saturation boost prevents the blur from looking washed out on dark backgrounds.
saturate(180%) blur(20px) + rgba(0,0,0,0.8)
Reis IA Adaptation
Apply directly to the navigation bar. The saturate(180%) trick is universally applicable to dark mode glassmorphism. No modification needed -- this technique is brand-neutral.
4. Invisible Transitions (200ms Default)
Transitions should be felt, not seen. Standard duration is 200ms with simple ease timing. Only color/opacity changes on hover -- no scale, no position shifts.
Hover me
color only, 200ms
Hover me
bg shift, 200ms
Hover me
border only, 200ms
Reis IA Adaptation
Adopt 200ms as the default transition duration across all components. Use color transitions for nav links and text. Use border-color transitions for cards. Reserve transform animations (scale, translate) for scroll-triggered entrance animations only.
5. Geist Typography System
Purpose-built font with Swiss precision. Variable weight (100-900), tight tracking at large sizes, loose tracking at small sizes. The font does most of the visual work.
Typography is design.
Large headlines with tight tracking, generous line-height on body text, and weight contrast between levels.
Reis IA Adaptation
Inter (our font) shares the same Swiss heritage as Geist. Apply the same tracking rules: -0.04em at 72px, -0.02em at 40px, 0 at 16px, +0.02em at 12px. Use weight 300 for light body text to create contrast with 600-700 headings.
