Key Typography Patterns

  • • Letter-spacing tightens as size increases: Display (-0.96px) > H1 (-0.64px) > H2 (-0.02em) > Body (0)
  • • Line-height decreases as size increases: Body (1.5-1.6) > H3 (1.2) > Display (1.1)
  • • Semibold (600) is the workhorse: subheadings, captions, buttons, card titles
  • • Captions use positive letter-spacing (+0.2px) for readability at small sizes
  • • No ALL CAPS except small labels/badges

Signature Elements

The 2-3 visual treatments that make Stripe instantly recognizable, with Reis IA adaptation notes.

Gradient Mesh Background
1. The Gradient Mesh / Wave Background
Multi-color organic gradient mesh behind hero sections. Extends beyond section bounds (-120px overshoot). Creates depth without competing with content. Too strongly associated with Stripe to copy directly.
Reis IA adaptation: Subtle geometric grid patterns at 3-5% opacity, or minimal gradient washes using gold/amber accent. Never the rainbow mesh — that IS Stripe.
Gradient Text
2. Knockout Gradient Text
Hero headlines with gradient fill via background-clip: text. Reveals from -20% to 50% over 1s, then loops at 5s. Background-size set to 1000% for smooth animation.
Reis IA adaptation: Use for a single accent word in hero headlines. Gold-to-white gradient. Slow 8s shimmer, not rainbow. Maximum ONE per page.
top: 3px radius: 16.5px bottom: 6px
3. Asymmetric Pill Button
Padding is intentionally asymmetric: 3px top, 6px bottom for optical centering. Border-radius 16.5px (not round). Hover is ONLY opacity — no color shift, no scale, no shadow.
Reis IA adaptation: Directly applicable. The asymmetric padding trick works universally. Opacity-only hover is elegant for gold CTAs.
Hover me scale(1.018)
Hover me 500ms ease
4. Card Scale + Shadow Depth
Exactly 1.018x scale (not 1.02, not 1.05). Shadow transitions from flat to deep over 500ms with cubic-bezier(0.7, 0, 0, 1). Makes cards feel like physical objects lifting off the surface.
Reis IA adaptation: Directly applicable. Use 1.015-1.02x scale with shadow depth. The 500ms duration with heavy easing is the key — feels premium because it's slow and weighted.

5. Section Background Alternation
Alternates between pure white (#fff) and barely-perceptible blue-gray (#f6f9fc). Creates visual rhythm and section separation without borders or dividers.
Reis IA adaptation: Alternate #000000 (pure black) and #0A0A0A (near-black) in dark mode. Even more subtle, fits premium feel perfectly.
