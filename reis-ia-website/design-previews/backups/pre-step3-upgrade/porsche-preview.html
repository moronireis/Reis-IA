Typography Scale

Porsche uses a proprietary typeface ("Porsche Next") with fluid sizing via clamp(). Only 3 font weights. Line height uses a mathematical formula: calc(6px + 2.125ex) -- scales proportionally across all sizes.

Display XX-Large

Heading X-Large

Heading Large

Text Small -- This is the recommended default body text size. It provides comfortable readability across all viewport sizes and maintains the visual elegance expected from a premium brand. The line-height formula ensures consistent optical spacing regardless of font size.

Signature Elements

The three defining characteristics that make Porsche's web presence feel uniquely premium. Each includes an interactive demonstration and notes on how to adapt the pattern for Reis IA.

1. Systematic Fluidity
Everything scales continuously via clamp() -- typography, spacing, grid gaps, safe zones. No jarring breakpoint jumps. The experience flows from mobile to ultrawide as a single continuous transformation.
Resize your browser to see fluid scaling:
Fluid Typography
font-size: clamp(1.6rem, 1.56vw + 1.29rem, 3.16rem)
Spacing tokens
scale with viewport
Reis IA Adaptation
Adopt the 6-token fluid spacing system directly. Replace Porsche's type scale with Inter-optimized clamp() values. Use the same calc(6px + 2.125ex) line-height formula -- test with Inter and adjust the 2.125 multiplier if needed (Inter's x-height differs from Porsche Next).
2. Extreme Restraint
The design system says "no" far more than it says "yes." Three font weights. One shadow value. No accent color in the UI. No gradients as decoration. No rounded pill buttons. What is absent defines the brand as much as what is present.

Reis IA Adaptation
Maintain the same restraint principle but add one deliberate exception: the muted gold accent (#C9A84C). Used only for CTAs, brand motifs, and key emphasis. Everything else follows Porsche's discipline -- no decorative gradients, no multiple shadows, no extra font weights. The gold accent should feel earned, not casual.
3. Precision Over Personality
Every value is engineered. The line-height formula, the custom easing curves, the non-standard breakpoint at 760px. Nothing is a default. The design system feels built by engineers, not decorated by designers. This is how precision becomes the personality.

Reis IA Adaptation
Apply the same principle of intentionality. Every Reis IA token should have a reason. Use warm-black (#0A0A0A) instead of blue-black -- our brand temperature is warm (gold accent), so the background should lean warm. Test the line-height formula with Inter. Match the 4-tier duration system exactly. The precision itself becomes a brand signal.
4. Image Hover -- Contained Scale
Images scale up by exactly 5% on hover using scale3d for GPU acceleration, contained within an overflow:hidden parent. The motion is subtle enough to feel luxurious -- seen, not noticed.
Hover the image:
scale3d(1.05, 1.05, 1.05)
.wrapper { overflow: hidden; }
.image {
  transition: transform 0.4s
    cubic-bezier(0.25, 0.1, 0.25, 1);
}
.wrapper:hover .image {
  transform: scale3d(1.05, 1.05, 1.05);
}
Reis IA Adaptation
Use this pattern for case study cards, portfolio images, and the founder bio photo. The 1.05 scale factor and 0.4s duration are already tuned for premium feel. Apply to image-heavy cards only -- text cards should use border/shadow hover instead.
