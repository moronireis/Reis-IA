Color System

02

Typography System

Inter Variable as primary, EB Garamond as display serif. 9-level title scale + 6 text sizes. Semibold (600) is the heaviest standard weight. text-wrap: balance on headlines; text-wrap: pretty on body.

04

Hardware-Adaptive Rendering

Linear checks navigator.hardwareConcurrency at page load. Machines with more than 4 cores get the "enhanced" class, enabling richer animations. Below 4 cores, simpler transitions are used.

Dot Grid Matrix Pattern

Linear's signature visual: 5x5 matrix of dots with discrete opacity steps. steps(1, end) creates binary appearance. Two animation families (3200ms agent + 2800ms upDown) create non-repeating patterns.

Component Patterns

Linear's components reflect "structure felt, not seen." Multiple button variants, dual-element hover patterns, and ultra-low-contrast underlines. Minimal decoration, maximum content focus.
