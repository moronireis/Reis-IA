[03] Rotating Border

Rotating Conic Gradient Border

Signature effect: CSS @property animates a conic-gradient angle over 6.4s. The card uses dual backgrounds -- padding-box for fill, border-box for the rotating gradient visible through a transparent 1px border.

LIVE6.4s linear infinite
Animated Border Card
Watch the subtle bright spot rotate around the card edge. Uses @property for smooth CSS Houdini angle animation.
click to replay
Implementation
@property --border-angle { syntax: '<angle>'; initial-value: 0deg; inherits: false; }
animation: rotateBorder 6.4s linear infinite;
background: linear-gradient(...) padding-box, conic-gradient(from var(--border-angle), ...) border-box; border: 1px solid transparent;

[07] Text Shimmer

Text Shimmer Effect

Academia Lendaria's ORIGINAL gray-to-white shimmer. A 13s ease-in-out gradient sweep creates an almost imperceptible shimmer on headlines. Gradient: #fff to #b8b8b8 and back. This is their native pattern -- NOT the Azure Whisper blue version.

[09] Hero Pattern

Inset Hero Pattern

The hero uses an inset pattern: 8px padding on outer wrapper, then a rounded-16px container inside. This creates a subtle "framed" effect with dark overlay gradient.
