02

Typography System

Letter-spacing tightens as size increases. Semibold (600) is the workhorse weight. Caption text uses positive letter-spacing (+0.2px) for readability at small sizes.

03

Spacing System

96-128px between sections. Most sites use 48-64px. Hero sections have 182-214px top padding. Content breathes. Stripe uses fluid spacing between 375px and 1112px viewport widths.

Micro-interaction Timing

Three tiers: 150ms for hovers, 250ms for UI changes, 850ms for dramatic entrances. Nothing bounces, nothing overshoots. Everything decelerates smoothly. Click to see the timing.
