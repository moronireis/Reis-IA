07 / Rationale
Design Decisions
Reference-informed reasoning behind each variation and the optical sizing system.
A. Mathematical Precision
Concept: True geometric center with maximum canvas utilization.

Reference: Apple's SF Symbols use a strict grid system where every icon fills its optical bounding box. Porsche centers marks with sub-pixel precision. Stripe's geometric wordmark uses mathematical ratios.

Changes: Intersection centered at exact y=32 (vs original ~y31). Bars pushed to y=8/56 for 48-unit height span. Gap 8 units for clean proportions.

Best at: All sizes. Recommended as the primary mark.
B. Structural Hierarchy
Concept: Variable stroke weights create focal hierarchy.

Reference: Linear's underline decoration uses 1.5px specifically. Apple's SF Symbols use different weights for different contexts. The variable-weight approach draws from typography's bold/regular contrast.

Changes: Bars 1.25px, diagonal 1.5px, intersection stubs 1.75px. The heavier stubs draw the eye to the gap — the mark's signature feature.

Best at: 48px+. Weight differences collapse at small sizes.
C. Micro-Rounded Path
Concept: Continuous path with natural round-join softening.

Reference: Porsche's motion system uses `cubic-bezier(0.25, 0.1, 0.25, 1)` for smoothness. Linear's stroke-draw animation reveals paths continuously. Both brands favor curves over sharp edges.

Changes: Z-shape (top bar, full diagonal, bottom bar) as a single path element. SVG `stroke-linejoin="round"` softens the two corners naturally without explicit border-radius.

Best at: 32px+. The rounded corners are subtle but add warmth.
D. Taller Proportions
Concept: Vertical elongation approaching the golden ratio.

Reference: Porsche's crest is taller than wide (approximately 1:1.4). The golden ratio (1:1.618) is used across Apple's layout system. Taller marks read as more elegant and authoritative.

Changes: Width narrowed to 28u (18-46), height expanded to 54u (5-59). Ratio 1:1.69. Creates a more hourglass-like silhouette.

Best at: 48px+. Narrow width can feel cramped at small sizes.
E. Maximum Gap
Concept: Widest possible gap for small-size legibility.

Reference: Apple's favicon designs prioritize recognition over detail. Morningside has zero brand iconography (our competitive gap to exploit). The mark must be instantly recognizable at 16px in a browser tab.

Changes: Gap expanded to 12 diagonal units (26,25.4)-(38,38.6). The open intersection becomes the dominant visual element.

Best at: 16-32px. At 128px+ the gap can feel too wide/disconnected.
Optical Sizing System
Concept: Each target size gets its own SVG with tuned parameters.

Reference: Apple's SF Symbols ship in 3 optical sizes (Small, Medium, Large) with different stroke weights. Porsche's icon system adjusts at every breakpoint. This is an industry-leading practice that most brands skip.

System: Stroke weight and gap width scale inversely with render size. At 16px: 3.5px stroke, 14u gap. At 128px: 1.25px stroke, 6u gap. This ensures the mark reads clearly at favicon scale and feels refined at hero scale.

Implementation: Use opt-16 for favicon, opt-24 for nav, opt-32 for inline, opt-48/64 for features, opt-128 for hero/watermarks.
