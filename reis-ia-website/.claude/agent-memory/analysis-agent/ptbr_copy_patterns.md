## Core insight
Brazilian B2B copy must combine emotional warmth with quantified outcomes. Direct translation from English always underperforms. The "Quero + outcome" CTA pattern is distinctly Brazilian and outperforms generic verbs.

**Why:** Researched from Ramper, Leadster, RD Station, Hotmart, HeroSpark, and multiple Brazilian copywriting sources in March 2026.
**How to apply:** When briefing the Copy Agent or reviewing PT-BR copy, use these patterns as the benchmark.

## Top CTA patterns
- "Quero [specific outcome]" — first-person desire, highest converting
- "Começar teste grátis" — low-friction SaaS entry
- "Solicitar diagnóstico gratuito" — consultative B2B
- "Falar com especialista" — bottom of funnel

## Top headline patterns
- Number + Outcome + Timeframe: "Aumente suas vendas em 120 dias..."
- Problem as Question: "Seu site está perdendo clientes por...?"
- Bold Claim + Objection Removal: "Organize o trabalho com facilidade. Comece sem custo."
- Curiosity gap: "Descubra como essa empresa dobrou o faturamento..."

## Key anti-patterns
- Direct translation from English sounds robotic ("unlock", "leverage", "crush your goals")
- Generic CTAs: "Clique aqui", "Saiba mais" alone, "Submit"
- Vague social proof (no company name, no numbers)
- Feature lists without benefit translation
- European Portuguese formality ("V.Sa.", passive voice)
