Design Principles
◼
Dark-First
Background #050505 como base. Superficies #0F0F11. Informacao emerge da escuridao.
◆
Neon Lime Accent
#D1FF00 como cor de destaque unica. CTAs, status ativo, dados positivos.
⌘
Monospace Voice
RobotoMono para labels. TASAOrbiter para titulos. Geist para corpo.
⊞
Grid Precision
Layouts baseados em grid com gap de 1px e border como separador.
⚠
HUD Language
Section dividers como cockpit readouts. Corner brackets. Hazard stripes.
◎
Data-Dense
Otimizado para exibir informacao densa — KPIs, rankings, metricas.
