# AIOX — Buttons Page

## Page Purpose
Button library documentation with production roles, variants, sizes, states, and usage guidance.

1. **Header** — "Button Library — Dark Cockpit Edition v1.0"

5. **04 Sizes** — Small, Medium, Large
6. **05 States** — Default, Loading, Disabled
7. **06 Combinations** — Example button pairs

## Production Roles
| Component | Purpose |
|-----------|---------|
| `SiteCta` | Primary marketing surface action |
| `Button` | Generic product/app-shell primitive |
| `PrimaryLink` | Inline editorial/navigation (low visual weight) |
| `CtaButton` | Legacy marketing primitive |

## Variants
- **Primary** — Lime background, dark text
- **Secondary** — Surface background, cream text, border
- **Ghost** — Transparent, cream text
- **Delete** — Destructive red variant

## Sizes
- **Small** — Compact for inline use
- **Medium** — Default
- **Large** — Hero/CTA prominence

## States
- **Default** — Normal interactive
- **Loading** — Spinner indicator
- **Disabled** — Reduced opacity, non-interactive

## Example Labels
"Book a Call", "Learn More", "Cancel", "Remove"

## Navigation Context
- Position: Design System > Components > Buttons (1.0)

## Key Design Decisions
- 4 distinct button components for different contexts (marketing vs product)
- 4 visual variants × 3 sizes × 3 states = comprehensive coverage
- Legacy support (CtaButton) maintained alongside modern components
