## Usage Rules (Verbatim)
1. "Always use 2px stroke width at all sizes"
2. "Round caps and round joins only"
3. "24px viewBox as canonical base"
4. "Stroke only, no fills"
5. "Use currentColor for inheritance"
6. "Minimum touch target: 44x44px"

## Key Design Decisions
- Stroke-only icon system (no fills)
- 2px consistent stroke weight
- 24px canonical viewBox
- currentColor for theme inheritance
- 44x44px minimum touch target (WCAG compliant)
- 4 size variants cover all use cases
