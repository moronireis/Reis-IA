# AIOX — Movimento (Strategic Brandbook) Page
URL: https://brand.aioxsquad.ai/brandbook/movimento

## Page Purpose
Comprehensive strategic brandbook with 13 numbered sections covering manifesto, purpose, archetype, positioning, BrandScript, truelines, naming, vocabulary, hero's journey, testimonials, visual identity, brand contract, and founders.

## Content Structure
1. **01 Manifesto** — "A IA e a seta. O X e seu." Core belief statement
2. **02 Purpose & Values** — Mission: "Devolver as pessoas o poder de criar." 6 core values
3. **03 Archetype & Personality** — Rebel/Outlaw 50%, Magician 35%, Explorer 15%. Morpheus analogy
4. **04 Positioning & Hierarchy** — Category: AI Orchestration Experience. Onlyness statement
5. **05 BrandScript** — Hero's Journey framework (hero, problem, guide, plan, failure, success)
6. **06 Truelines & Taglines** — 5 tagline variants with usage contexts
7. **07 Naming Architecture** — A→I→O→X semantic breakdown. 89% naming score. Sub-brands
8. **08 Vocabulary & Tone** — Power words (use), banned words (avoid), 5 tone dimensions
9. **09 Hero's Journey** — 6-stage transformation arc with user quotes
10. **10 Testimonials** — Measurable results + qualitative evidence
11. **11 Visual Identity** — Marks (triangle, joystick), full color palette, typography
12. **12 Brand Contract** — Brand promises + creator demands + manifesto
13. **13 Founders** — Alan Nicolas (Morpheus Humano) + Pedro Valerio (Arquiteto Sistemico)

## HTML Structure
- Long-form scrollable page with 13 major sections
- Sidebar navigation for section jumping (Fundamentos, Estrategia, Identidade Verbal, Jornada & Prova, Identidade Visual, Compromisso)
- Each section has numbered label and content blocks

## CSS Patterns
- Fluid typography: clamp(4rem, 10vw, 8rem)
- Layout: Flexbox columns with min-h-screen
- Variables: --void (#000000), --terminal (#050505), --surface (#0F0F11), --olive (#1C1E19)
- Light tokens: --cream (#F4F4E8), --warm-white (#FFFFED), --white (#FFFFFF)
- Accents: --kinetic-limon (#D1FF00), --signal-blue (#0099FF), --flare (#ED4609)
- Fonts: Geist (headlines/body), RobotoMono (code/terminal)

## JavaScript Interactions
- Theme toggle: LimeGold selector
- Section-based sidebar navigation
- React Server Components streaming

## Components Used
- BrandbookSiteNav, BrandbookThemeRoot, BrandbookTracking

## Navigation Context
- Position: Second item in Brandbook section (1.0 Movimento)
- Primary nav: Guidelines > Movimento > Pitch > Workspace
- Contains deepest brand strategy content

## Key Design Decisions
- Most content-heavy page in the entire system
- Bilingual (Portuguese primary, English secondary)
- Hero's Journey framework as brand narrative structure
- Matrix/Morpheus as central brand metaphor
- Two distinct brand voices: AIOX (cold/minimal) vs Alan (warm/expansive)
- Full color palette defined here: VOID, TERMINAL, SURFACE, OLIVE, CREAM, WARM WHITE, KINETIC LIMON, SIGNAL BLUE, FLARE
