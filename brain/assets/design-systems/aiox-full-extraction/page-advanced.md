# AIOX — Advanced UI Page

## Page Purpose
Advanced UI component library covering tabs, accordion, and steppers.

## Content Structure
1. **Header** — "Advanced UI Library V1.0 // Dark Cockpit Edition"
2. **Tabs** — Project overview with metrics and status dashboard
3. **Accordion** — FAQ content (What is AIOX?, implementation, programming, security)
4. **Steppers** — Horizontal + Vertical variants

## Components

### Tabs
- Project overview with key metrics
- Status dashboard view
- Configuration and logging views

### Accordion
FAQ items with expand/collapse (arrow rotation):
- What is AIOX?
- Implementation details
- Programming requirements
- Security considerations

### Steppers
**Horizontal**: Discovery → Architecture → Implementation → QA → Deploy
**Vertical**: Onboarding → Data Integration → Training → Go Live

Step states: completed (checkmark), active, upcoming

## Navigation Context
- Position: Design System > Components > Advanced (11.0)

## Key Design Decisions
- Tabs for multi-view content
- Accordion for collapsible content (FAQ pattern)
- Steppers for process visualization (both orientations)
- Completion indicators (checkmark for done, highlight for active)
